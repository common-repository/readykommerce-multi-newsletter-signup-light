== Description ==

Integrate MailChimp in your WordPress site with AJAX powered customizable multiple  form without touching code.

= Features: =
------------
- Easily create a highly customizable sign-up form. No HTML knowledge required.
	- Show/Hide form fields.
	- Set validation for form fields.
- Show it in your posts, pages or any custom post type
	- Widget support.
	- Shortcode support.
- Uses the API, superfast, reliable and integrated with MailChimp. 
- Visitors do not have to leave your website to subscribe. 
- (Multiple) AJAX powered sign-up forms (no page reload after submitting).
- Mobile optimized/responsive form.
- Configuring or add users to multiple email lists.
- Multiple campaigns in one website.
- Multiple newsletters support in one page with multiple lists support
- Theme compatible.


= Features in full version: =
-----------------------------
- Customize option to change message/placeholder/label etc.
- Design beautiful sign-up forms from admin panel. No CSS knowledge required
- Custom css support
- Google Analytics integration
- Subscribe to download
- Subscribe to unlock article.
- Show popup to your visitors with a promo
- Easily add YouTube or Vimeo videos to popup subscription form
- Custom notification system to admin
- Language support.

== Installation ==

1. Upload `nl_plugin-light` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure your MailChimp API and form in 'Settings > readyKommerce Newsletter – Light' menu in WordPress
1. Place `NL Signup` widget in your sidebar or use `[nl_mc_form]` shortcode in your post/page.
1. For more details please download [official user guide](http://cdn.readykommerce.com/readyKommerce-Newsletter-Plugin-UserGuide.pdf)

== Frequently asked questions ==

= Where i can get MailChimp API? =

Login to your [MailChimp](http://mailchimp.com) account and get your api from [here](https://us6.admin.mailchimp.com/account/api/) or read [Where can I find my API Key?](http://kb.mailchimp.com/article/where-can-i-find-my-api-key/)

= How can I find MailChimp List ID? =

Read MailChimp [official guide](http://kb.mailchimp.com/article/how-can-i-find-my-list-id/) to find List ID.
